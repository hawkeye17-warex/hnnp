# NearID Admin Console

This is the web dashboard for NearID organization administrators. It is a React + TypeScript single-page app built with Vite.

## Getting started
1) Install dependencies
```bash
cd admin-console
npm install
```
2) Run the dev server
```bash
npm run dev
```
The app runs on http://localhost:5173 by default.

## Scripts
- `npm run dev` — start Vite dev server
- `npm run build` — type-check and build for production
- `npm run preview` — preview the production build

## Notes
- All source lives in `admin-console/src`.
- Customize components in `src/App.tsx` and styles in `src/styles.css`.
